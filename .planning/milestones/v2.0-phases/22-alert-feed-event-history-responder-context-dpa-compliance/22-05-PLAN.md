| T-22-05-02 | T (Tampering) | Operator A re-ACKs event that Operator B already ACK'd | mitigate | Controller abort_if on existing acknowledged_at → 409 Conflict. Wave 3 Plan 22-06 Vue layer handles 409 gracefully (removes card silently). |
