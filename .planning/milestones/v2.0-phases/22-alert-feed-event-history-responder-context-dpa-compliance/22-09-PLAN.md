```markdown
# FRAS Operator Training — IRMS DPA Compliance

## Role Matrix
| Role | /fras/alerts | /fras/events | Manage Cameras | Manage Personnel | Promote Event | View Scene Image |
|------|:-:|:-:|:-:|:-:|:-:|:-:|
| Admin | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ |
| Supervisor | ✓ | ✓ | ✓ | ✓ | ✓ | ✓ |
| Operator | ✓ | ✓ | ✗ | ✗ | ✓ | ✓ |
| Dispatcher | ✗ | ✗ | ✗ | ✗ | ✗ | ✗ |
| Responder | ✗ | ✗ | ✗ | ✗ | ✗ | ✗ |

## ACK vs Dismiss
- **Acknowledge**: "I'm handling this". No reason required. Removes card from all operator feeds.
- **Dismiss**: "Not actionable / false positive". Reason required (False match / Test event / Duplicate / Other). Persisted on recognition_events.

## When to Promote to Incident
Promote is available on non-Critical events (Warning severity or previously-dedup'd Critical). Requires a priority (P1–P4) and a reason (8–500 chars). Creates a tracked Incident tied to the original recognition event.

## Scene Image Access Restrictions
- Operators/Supervisors/Admins can view the scene image via signed 5-minute URL. Every view is logged.
- Responders CANNOT access the scene image — the backend never serves it to responder browsers.
- Citizens (DPA subjects) have no direct access; they exercise their rights via DPO contact.

## Signed URL Expiry
Every face + scene image URL expires server-side after 5 minutes. Reload the page to regenerate. Browser or screen captures beyond the window are no longer technical requests (they are a human re-record — a DPA reviewer concern, not a tech concern).

## Retention Purge
- Scene: 30 days after captured_at
- Face: 90 days after captured_at
- Exception: events linked to active (non-resolved, non-cancelled) incidents are protected until case closes.
- Access log: 2 years.
- Daily at 02:00 Asia/Manila via `php artisan fras:purge-expired`.
- Dry-run available for legal pre-go-live review: `php artisan fras:purge-expired --dry-run`.
```

```php
class FrasDpaExport extends Command
{
    protected $signature = 'fras:dpa:export {--doc=all : pia|signage|training|all} {--lang=en : en|tl}';
    protected $description = 'Export DPA documentation (PIA, signage, operator training) to PDF via dompdf';

    public function handle(): int
    {
        $doc = $this->option('doc');
        $lang = $this->option('lang');
        $docs = $doc === 'all' ? ['pia', 'signage', 'training'] : [$doc];
        $outDir = storage_path('app/dpa-exports/' . now()->format('Y-m-d'));
        if (!is_dir($outDir)) { mkdir($outDir, 0755, true); }

        $converter = new \League\CommonMark\GithubFlavoredMarkdownConverter(['html_input' => 'strip']);

        foreach ($docs as $d) {
            $mdFile = match ($d) {
                'pia' => base_path('docs/dpa/PIA-template.md'),
                'signage' => base_path('docs/dpa/signage-template' . ($lang === 'tl' ? '.tl' : '') . '.md'),
                'training' => base_path('docs/dpa/operator-training.md'),
                default => throw new \InvalidArgumentException("Unknown doc: {$d}"),
            };
            $md = file_get_contents($mdFile);
            $html = (string) $converter->convert($md);
            $pdf = \Barryvdh\DomPDF\Facade\Pdf::loadView('dpa.export', [
                'content' => $html,
                'title' => ucfirst($d),
            ]);
            $out = "{$outDir}/{$d}-{$lang}.pdf";
            $pdf->save($out);
            $this->info($out);
        }

        return self::SUCCESS;
    }
}
```

class FrasLegalSignoff extends Command
{
    protected $signature = 'fras:legal-signoff {--signed-by= : Signatory name} {--contact= : Email or phone} {--notes=}';
    protected $description = 'Record CDRRMO legal sign-off for Phase 22 DPA compliance';

    public function handle(): int
    {
        $signedBy = $this->option('signed-by');
        $contact = $this->option('contact');
        $notes = $this->option('notes');

        if (!$signedBy || !$contact) {
            $this->error('--signed-by and --contact are required.');
            return self::INVALID;
        }

        $signoff = \App\Models\FrasLegalSignoff::create([
            'signed_by_name' => $signedBy,
            'contact' => $contact,
            'signed_at' => now(),
            'notes' => $notes,
        ]);

        // Append to VALIDATION.md
        $validationPath = base_path('.planning/phases/22-alert-feed-event-history-responder-context-dpa-compliance/22-VALIDATION.md');
        if (file_exists($validationPath)) {
            $iso = $signoff->signed_at->toIso8601String();
            $appendLine = "\n- [x] CDRRMO legal sign-off recorded via `php artisan fras:legal-signoff` (DPA-07) — signed by {$signedBy} on {$iso}\n";
            file_put_contents($validationPath, $appendLine, FILE_APPEND);
        }

        $this->info("Legal sign-off recorded: {$signedBy} at {$signoff->signed_at->toIso8601String()}");
        return self::SUCCESS;
    }
}
