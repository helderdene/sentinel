<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Test fixture file ↔ production broadcast payload | The committed JSON is the regression oracle for what operators/responders/dispatchers will receive. Any drift between fixture and `broadcastWith()` output is an information-disclosure or contract-drift signal. |
| Factory-generated values ↔ deterministic test state | Factories use `fake()` generators that produce different values every run unless pinned. Non-determinism here masquerades as a "real" regression. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
