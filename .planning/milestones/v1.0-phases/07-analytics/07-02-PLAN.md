AnalyticsController endpoints:
- GET /analytics/dashboard -- Inertia page with props: kpis, timeSeries, filters, filterOptions
- GET /analytics/heatmap -- Inertia page with props: density, geojson, filters, filterOptions
- GET /analytics/heatmap/barangay/{barangay} -- JSON: { top_types, priority_breakdown, total, name }
- GET /analytics/reports -- Inertia page with props: reports (paginated)
- GET /analytics/reports/{report}/download -- file stream
- POST /analytics/reports/generate -- dispatch report job

KPI data shape (from AnalyticsService):
```typescript
interface KpiMetrics {
    avg_response_time_min: number | null;
    avg_scene_arrival_time_min: number | null;
    resolution_rate: number;
    unit_utilization: number;
    false_alarm_rate: number;
}

interface KpiTimeSeries {
    metric: string;
    data: Array<{ date: string; value: number }>;
}
```

Barangay density shape:
```typescript
interface BarangayDensity {
    barangay_id: number;
    name: string;
    incident_count: number;
}
```

GeoJSON: Standard FeatureCollection with barangay id, name in properties.

Filter options:
```typescript
interface FilterOptions {
    incidentTypes: Array<{ id: number; name: string; code: string }>;
    barangays: Array<{ id: number; name: string }>;
}
```

- Design system tokens from Phase 8: DM Sans, Space Mono fonts, color-mix() for opacity
