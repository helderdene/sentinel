**For all 3 admin table pages (Users.vue, Barangays.vue, IncidentTypes.vue), apply the design system data table pattern:**

1. **Table wrapper:** Replace `overflow-hidden rounded-lg border` with `overflow-hidden rounded-[7px] border border-border bg-card shadow-[var(--shadow-1)]`

2. **Column headers (thead):** Replace `border-b bg-muted/50` with `border-b border-border bg-card`:
   - Each `<th>` gets: `class="px-4 py-3 font-mono text-[9px] font-bold tracking-[2px] text-t-text-faint uppercase"` (replacing `font-medium`)

3. **Data rows:** Replace `hover:bg-muted/30` with `hover:bg-accent transition-colors`
   - Add `border-b border-border` to each `<tr>` instead of using `divide-y` on tbody
   - Remove `divide-y` from `<tbody>`

4. **Data cells:**
   - Name/title cells: `text-foreground font-medium` (DM Sans, inherited)
   - Secondary data cells: `text-muted-foreground` (maps to t-text-dim)
   - ID/code cells: add `font-mono text-[10px] text-t-text-faint` for machine-readable data

5. **Empty state:** Replace `text-muted-foreground` in empty state divs with `text-t-text-faint` and add `shadow-[var(--shadow-3)]` if it's a separate card

6. **Users.vue specifically -- Role badge colors:** Replace the hardcoded roleColors map with design system role colors:
```typescript
const roleColors: Record<string, string> = {
    admin: 'bg-[color-mix(in_srgb,var(--t-role-admin)_12%,transparent)] text-t-role-admin',
    dispatcher: 'bg-[color-mix(in_srgb,var(--t-accent)_12%,transparent)] text-t-accent',
    operator: 'bg-[color-mix(in_srgb,var(--t-role-operator)_12%,transparent)] text-t-role-operator',
    responder: 'bg-[color-mix(in_srgb,var(--t-online)_12%,transparent)] text-t-online',
    supervisor: 'bg-[color-mix(in_srgb,var(--t-role-supervisor)_12%,transparent)] text-t-role-supervisor',
};
```
This uses color-mix() for background tints (established Phase 8 pattern) with the design system role token colors. Remove the `dark:` variants -- color-mix() with CSS variables automatically adapts.

Run `npm run lint` and `npm run format` after all changes.
