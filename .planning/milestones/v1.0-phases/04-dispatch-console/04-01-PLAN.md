Use `ST_DWithin(coordinates, ST_SetSRID(ST_MakePoint(?, ?), 4326)::geography, ?)` for filtering and `ST_Distance(coordinates, ST_SetSRID(ST_MakePoint(?, ?), 4326)::geography)` for sorting.     4. **DispatchConsoleController** with 6 actions:
       - `show()`: Inertia render 'dispatch/Console' with incidents (TRIAGED + DISPATCHED + EN_ROUTE + ON_SCENE, with incidentType, barangay, assignedUnits loaded), units (all, with coordinates), agencies (with incidentTypes). Include session metrics: totalIncidents (non-resolved count), activeIncidents (DISPATCHED + EN_ROUTE + ON_SCENE count), criticalIncidents (P1 active count), unitsAvailable (AVAILABLE count), unitsTotal (non-OFFLINE count), **averageHandleTime** -- compute as avg(TIMESTAMPDIFF in minutes between resolved_at and created_at) for incidents resolved today (where resolved_at >= today midnight). Return null if no resolved incidents today. Use `Incident::query()->where('status', IncidentStatus::Resolved)->whereDate('resolved_at', today())->selectRaw('AVG(EXTRACT(EPOCH FROM (resolved_at - created_at)) / 60) as avg_handle_time')->value('avg_handle_time')` (adjust for PostgreSQL syntax).        - `assignUnit(AssignUnitRequest, Incident)`: Validate unit is AVAILABLE. Attach to incident_unit pivot (assigned_at, assigned_by). Update unit status to DISPATCHED. Dispatch IncidentStatusChanged if status changed. Return JSON success.
       - `unassignUnit(UnassignUnitRequest, Incident)`: Set unassigned_at on pivot. If unit has no other active assignments, set unit status back to AVAILABLE. Create timeline entry (event_type: unit_unassigned). Dispatch UnitStatusChanged. Return JSON success.

Update incident status and corresponding timestamp field. Create timeline entry. Dispatch IncidentStatusChanged. If RESOLVED, set resolved_at. Return JSON success.
       - `requestMutualAid(MutualAidRequest, Incident)`: Create timeline entry with event_type: mutual_aid_requested, event_data with agency_id, agency_name, notes. Dispatch MutualAidRequested broadcast event. Return JSON success.
