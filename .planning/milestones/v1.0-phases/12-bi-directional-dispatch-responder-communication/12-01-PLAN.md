<objective>
Refactor the MessageSent event from user-level to incident-level channel broadcasting, add incident channel authorization, create a dispatch-side sendMessage endpoint, and update the responder sendMessage to use the new event constructor.

Purpose: The current messaging system broadcasts on `user.{recipientId}` which only supports point-to-point communication. This plan shifts to `incident.{id}.messages` for true group chat where all participants (dispatch + all assigned units) see all messages. Additionally broadcasting on `dispatch.incidents` ensures all online dispatchers see messages for any active incident.

Output: Refactored MessageSent event, incident channel auth, dispatch sendMessage endpoint, updated responder sendMessage, 4 test files covering all behaviors.
</objective>
