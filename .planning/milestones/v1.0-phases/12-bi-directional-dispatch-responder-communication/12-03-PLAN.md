<objective>
Migrate the responder messaging channel subscription from `user.{id}` to `incident.{id}.messages` and update the ChatTab UI to display group chat sender identification (unit callsign + name).

Purpose: The current responder messaging subscribes to `user.{userId}` which only receives messages addressed to that specific user. Migrating to the incident-level channel enables true group chat where all participants (dispatch + all assigned units) see all messages. The ChatTab UI needs updated sender display to identify who sent each message in multi-unit incidents.

Output: Updated useResponderSession with incident-level channel subscription, updated MessagePayload/IncidentMessageItem types, updated ChatTab with unit callsign display.
</objective>
