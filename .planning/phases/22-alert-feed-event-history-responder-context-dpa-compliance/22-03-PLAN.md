| Responder → scene image | Three-layer exclusion; this plan delivers layer 1 (controller role gate) |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-22-03-01 | I (Info disclosure) | Responder fetching scene image via direct signed URL | mitigate | FrasEventSceneController role gate `[Operator,Supervisor,Admin]` — responder → 403. | T-22-03-02 | R (Repudiation) | Audit log write fails silently, image served without record | mitigate | DB::transaction wrap; any DB error throws → 500 response, stream never returned. Test simulates FK violation, asserts 500. |
| T-22-03-03 | T (Tampering) | Signed URL replayed past 5-min TTL | mitigate | Laravel's `URL::temporarySignedRoute` HMAC-SHA256 + server-clock expiry check; test asserts 403 on expired. |
