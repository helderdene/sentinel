>
> Phase 22 introduces **ZERO new CSS tokens.** Every color, spacing, type, and interaction choice binds to the frozen Sentinel tokens already shipping in `resources/css/app.css` (v1.0 Phase 10/14 + Phase 20 additions + Phase 21's single `--t-ch-fras` addition). Severity badges, ACK/Dismiss colors, POI accordion chip tints, privacy notice prose ‚Äî all derive from existing tokens.
