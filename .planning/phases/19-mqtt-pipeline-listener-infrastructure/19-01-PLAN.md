```php
public static function fromEvent(int $personType, int $verifyStatus): self
{
    if ($personType === 1) { return self::Critical; }
    if ($verifyStatus === 2) { return self::Warning; }
    if (in_array($verifyStatus, [0, 3], true)) { return self::Ignored; }
    return self::Info;
}
```

Downstream effect acceptable per Phase 22 DPA scope (Info events stay history-only, never broadcast).
