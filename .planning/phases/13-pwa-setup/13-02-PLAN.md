       - Constructor: `public Incident $incident`
       - `via()` returns `[WebPushChannel::class]`
       - `toWebPush()` returns WebPushMessage with:
         - title: "P1 CRITICAL"
         - body: "{incident_no} - {incidentType name} at {location_text}"
         - icon: "/pwa-192x192.png"
         - tag: "p1-{incident.id}"
         - data: `['url' => '/dispatch', 'incident_id' => incident.id]`
         - options: `['TTL' => 300, 'urgency' => 'high']`

    3. Create `app/Notifications/AckTimeoutNotification.php`:
       - Constructor: `public Incident $incident, public string $unitId`
       - `via()` returns `[WebPushChannel::class]`
       - `toWebPush()` returns WebPushMessage with:
         - title: "Assignment Not Acknowledged"
         - body: "{incident_no} requires acknowledgement"
         - icon: "/pwa-192x192.png"
         - tag: "ack-timeout-{incident.id}"
         - data: `['url' => '/assignment', 'incident_id' => incident.id]`
         - options: `['TTL' => 120, 'urgency' => 'high']`

    4. Create `app/Listeners/SendAssignmentPushNotification.php`:
       - Handles `AssignmentPushed` event
       - Finds User by `$event->userId`
       - Sends `AssignmentPushedNotification` via `$user->notify()`
       - Also dispatches `CheckAckTimeout` job with 90-second delay:
         `CheckAckTimeout::dispatch($event->incident->id, $event->unitId, $event->userId)->delay(now()->addSeconds(90))`

    5. Create `app/Listeners/SendP1PushNotification.php`:
       - Handles `IncidentCreated` event
       - Guard: only proceed if `$event->incident->priority === IncidentPriority::P1`
       - Query all users with dispatcher role: `User::where('role', UserRole::Dispatcher)->get()`
       - Also include operators and supervisors (they monitor incidents): `User::whereIn('role', [UserRole::Dispatcher, UserRole::Operator, UserRole::Supervisor])->get()`
       - Send `P1IncidentNotification` via `Notification::send($users, new P1IncidentNotification($event->incident))`

    6. Create `app/Jobs/CheckAckTimeout.php`:
       - Implements `ShouldQueue`
       - Constructor: `public int $incidentId, public string $unitId, public int $userId`
       - `handle()`:
         a. Query `incident_unit` pivot where incident_id = incidentId AND unit_id = unitId AND acknowledged_at IS NULL
         b. If found (still unacknowledged), find User by userId and send `AckTimeoutNotification`
         c. If acknowledged_at is set, do nothing (assignment was acknowledged in time)
       - Use `DB::table('incident_unit')` for the pivot query (no model for pivot)

    7. Create `app/Http/Controllers/PushSubscriptionController.php`:
       - `store(Request $request)`: Validate endpoint, public_key, auth_token, content_encoding. Call `$request->user()->updatePushSubscription()` (from HasPushSubscriptions trait) with endpoint, public_key, auth_token, content_encoding. Return 201 JSON.
       - `destroy(Request $request)`: Validate endpoint. Call `$request->user()->deletePushSubscription($endpoint)`. Return 204.
       - Both methods require `auth` middleware.

    8. Register routes in `routes/web.php`:
       Add inside the auth middleware group:
       ```php
       Route::post('/push-subscriptions', [PushSubscriptionController::class, 'store'])->name('push-subscriptions.store');
       Route::delete('/push-subscriptions', [PushSubscriptionController::class, 'destroy'])->name('push-subscriptions.destroy');
       ```

    9. Register event listeners in `app/Providers/AppServiceProvider.php` `boot()`:
       ```php
       use Illuminate\Support\Facades\Event;
       use App\Events\AssignmentPushed;
       use App\Events\IncidentCreated;
       use App\Listeners\SendAssignmentPushNotification;
       use App\Listeners\SendP1PushNotification;

       Event::listen(AssignmentPushed::class, SendAssignmentPushNotification::class);
       Event::listen(IncidentCreated::class, SendP1PushNotification::class);
       ```
       Add these lines in the `boot()` method (not inside any sub-method -- add a new helper method `configureEventListeners()` following existing pattern, or add directly in boot).

    10. Run `vendor/bin/pint --dirty --format agent` to format all new PHP files.
