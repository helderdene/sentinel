  <action>
    1. Create src/composables/useApi.ts:
       - Export function useApi() returning { get, post, loading, error }
       - Base URL: detect from import.meta.env.VITE_API_URL or default to '' (Vite proxy handles /api)
       - get<T>(path: string): Promise<T> -- fetch with GET, parse JSON, handle errors
       - post<T>(path: string, body: Record<string, unknown>): Promise<T> -- fetch with POST, Content-Type: application/json, Accept: application/json
       - Both methods set loading ref, catch errors to error ref
       - Return 422 validation errors in structured format for form display

    2. Create src/composables/useGeolocation.ts:
       - Export function useGeolocation()
       - Reactive refs: latitude, longitude, status ('idle'|'requesting'|'granted'|'denied'), error
       - requestLocation(): Promise<boolean> -- checks navigator.geolocation support, calls getCurrentPosition with enableHighAccuracy:true, timeout:10000, maximumAge:60000
       - On success: set lat/lng/status='granted', return true
       - On failure: set status='denied'/error message, return false

    3. Create src/composables/useReportStorage.ts:
       - Storage key: 'irms-citizen-reports'
       - getReports(): StoredReport[] -- parse from localStorage
       - addReport(report: StoredReport): void -- unshift, cap at 50, write back
       - updateReportStatus(token: string, status: string): void -- find by token, update status
       - removeReport(token: string): void -- filter out by token
       - Return all methods plus a reactive reports ref that stays in sync

    4. Create src/components/BottomNav.vue:
       - Three tabs: Home (house icon), My Reports (list icon), About (info circle icon)
       - Use SVG icons inline (matching prototype's I.home, I.list, I.info icons)

       - Fixed bottom bar: h-20, white bg, border-top, flex items
       - Uses router-link or programmatic router.push on click
       - Determines active tab from current route path

    5. Create src/components/PriorityBadge.vue:
